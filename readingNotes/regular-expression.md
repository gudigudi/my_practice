### 元字符
| 元字符 | 含义         |
| ------ | ------------ |
| ^      | 串的开始     |
| $      | 串的结束     |
| *      | 零到多次匹配 |
| +      | 一到多次匹配 |
| ?      | 零到一次匹配 |


### 转义字符
| 字符     | 含义           |
| -------- | -------------- |
| 字符本身 | 匹配字符本身   |
| \r       | 匹配回车       |
| \n       | 匹配换行       |
| \t       | 制表符         |
| \f       | 换页           |
| \x#      | 匹配十六进制数 |
| \cX      | 匹配控制字符   |


### 范围及重复
| 标识符 | 含义                              |
| [...]  | 在集合中的任一个字符              |
| [^...] | 不在集合中的任一个字符            |
| .      | 除\n之外的任一个字符              |
| \w     | 所有的单字, 包括字母,数字及下划线 |
| \W     | 不包括所有的单字,\w的补集         |
| \s     | 所有的空白字符,包括空格,制表符    |
| \S     | 所有的非空白字符                  |
| \d     | 所有的数字                        |
| \D     | 所有的非数字                      |
